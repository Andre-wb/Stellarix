# `android/app/src/test/java/` — JVM Unit Tests

Kotlin test sources — mirrors the `sol.vortexx.android.*` package layout under `../../main/java/`. One test class per production class, JUnit 5 + MockK, coroutine-aware (`runTest`). Runs on the JVM via `./gradlew test`.

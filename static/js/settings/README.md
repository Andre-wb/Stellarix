# `static/js/settings/` — Settings UI Modules

Per-pane settings modules used by `/settings` and its in-modal variant.

## Modules

- `profile.js` — profile editor (avatar upload, display name, status, bio).
- `privacy.js` — privacy toggles (hide online, hide last-seen, receipt policy).
- `security.js` — 2FA, passkeys, device list, active sessions, panic mode.
- `notifications.js` — per-room mute, quiet hours, push preferences.
- `themes.js` — app theme (dark / light / auto), accent picker.
- `i18n.js` — language switcher, RTL toggle.
- `backup.js` — key-backup vault UI.
- `about.js` — version, integrity status, credits.

## Loaded from

`../../../templates/screens/settings.html` and `../../../templates/modals/settings.html`.
